Get testimonials using the QuickVouch WordPress plugin!

== Description ==

Collect valuable sales boosting customer [testimonials](http://quickvouch.com). 

**What does this plugin do?**
 
This plugin enables your WordPress powered site to collect testimonials without having to write any code. 

**Why do I need testimonials?** 

Don't you ask for recommendations before trusting a person, product or service? Testimonials have been an essential part of any business since the beginning of time. Best of all they are usually **FREE** and all you have to do is ask for them. QuickVouch makes collecting testimonials super simple using your WordPress site.

**What do I need to do?** 

Install this plugin, activate it and enter your QuickVouch account information. You will need to create a **[QuickVouch account](http://quickvouch.com)**.

**Go here to create your QuickVouch account:** [http://quickvouch.com/](http://quickvouch.com/)

== Installation ==

**New QuickVouch users**, start here: 

1. Create a QuickVouch account here. http://quickvouch.com/sign-up
2. Set up your account, takes less than 5 minutes.
3. Download the plugin from the WordPress directory and install it. 
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Enter your QuickVouch account info and save your settings!
6. That's it!
7. One extra line here… we are superstitious… :)

 
**Existing QuickVouch users**, start here (simple as 1,2,3): 

1. Download the plugin from the WordPress directory and install it. 
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your QuickVouch account info and save your settings!
